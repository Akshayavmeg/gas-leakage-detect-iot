# Smart Gas Leakage Detection System

## Components Used
- Arduino Uno
- MQ2 Gas Sensor
- Buzzer
- LED
- Breadboard & Wires

## Description
This project detects gas leakage using MQ2 sensor.
If gas level exceeds threshold:
- LED turns ON
- Buzzer alerts user

## How to Run
1. Upload code to Arduino
2. Open Serial Monitor
3. Adjust threshold if needed

## Future Improvements
- IoT alerts (WiFi/GSM)
- LCD display
- Automatic gas shutoff
